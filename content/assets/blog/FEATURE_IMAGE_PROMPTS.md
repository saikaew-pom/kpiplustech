# Feature image prompt set

All 24 new images were created with the built-in image generation tool using the `photorealistic-natural` use case.

## Shared visual direction

> Create a 16:9 editorial feature image for a Thailand-first B2B technology article. Use premium, natural documentary photography with authentic Thai business or hospitality settings, real people and practical devices. Use deep navy, white and subtle light-blue accents. Compose a wide landscape image with a clear subject and useful negative space. Do not include logos, readable interface text, watermarks, sci-fi holograms, stereotypes or exaggerated emotions.
